# Лабораторная работа: Решение биквадратного уравнения

## Описание
Программа решает биквадратное уравнение вида:  
**Ax⁴ + Bx² + C = 0**

## Структура проекта
- `biquadratic_oop.py` - решение с использованием ООП
- `biquadratic_proc.py` - процедурное решение
- `tests.py` - модульные тесты
- `test_cases.txt` - примеры тестовых данных
- `README.md` - этот файл

## Использование

### ООП версия:
```bash
python biquadratic_oop.py
python biquadratic_oop.py 1 -5 4  # с аргументами командной строки
