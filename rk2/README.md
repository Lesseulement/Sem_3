# Рубежный контроль №2 по Python

## Описание проекта
Проект содержит рефакторинг программы из РК1 с добавлением модульных тестов.

## Структура проекта
- `refactored.py` - рефакторированный основной код
- `test_refactored.py` - модульные тесты
- `run_tests.py` - скрипт для запуска тестов
- `original_rk1.py` - исходный код из РК1 (опционально)

## Запуск программы
```bash
python refactored.py
